# Setarys

Lightweight network diagnostics and browser privacy analysis toolkit.

## Overview

Setarys is a static web-based tool designed to provide essential network and privacy diagnostics directly from the browser.

No backend processing, no data collection — all tests run client-side using standard web APIs and public endpoints.

## Features

- IP detection and geolocation
- DNS resolution checks (DoH)
- WebRTC leak detection
- Latency measurement
- HTTP header inspection
- Browser environment & fingerprint overview

## Principles

- **Client-side first** — all logic runs in your browser  
- **No data collection** — no analytics, no tracking  
- **Minimal stack** — no frameworks, no dependencies  
- **Open source** — fully auditable  

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript  
- WebRTC (RTCPeerConnection API)  
- Performance API  
- Public APIs (Cloudflare, ipapi, httpbin)

## Usage

Simply open the website and run the desired tools.  
No installation or configuration required.

## Development

This project intentionally avoids build tools and frameworks.
